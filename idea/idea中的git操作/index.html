大家在使用 Git 时，都会选择一种 Git 客户端，在 IDEA 中内置了这种客户端，可以让你不需要使用 Git 命令就可以方便地进行操作，本文将讲述 IDEA 中的一些常用 Git 操作。

环境准备

使用前需要安装一个远程的 Git 仓库和本地的 Git 客户端。
